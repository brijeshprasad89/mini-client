new Client(opts)

Builds a new remote or local client

Source:
  • client.js, line 40

Local clients need full service definition, including opts.name and opts.version. Local API could be exposed:

  • directly using opts.name & opts.init
  • with groups using opts.groups and opts.groupOpts`

For remote clients, only opts.remote is required
